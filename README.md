# Парсер политических партий Минюста РФ

Программа скачивает HTML-страницу с перечнем политических партий с сайта Министерства юстиции РФ и извлекает для каждой партии название и ссылку на документ (устав, регистрационные сведения и т.п.). Результат сохраняется в файл `parties.json` в виде списка словарей `{ "name": "...", "doc_url": "..." }`.

## Источник данных

Данные берутся с официального сайта Минюста РФ, страница:

https://minjust.gov.ru/ru/pages/politicheskie-partii/

При запуске скрипт сам скачивает HTML этой страницы с помощью библиотеки `requests`.

## Установка

Рекомендуется использовать виртуальное окружение.
